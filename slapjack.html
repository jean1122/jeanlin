This project is a two-player Slapjack game designed with C and ARM Assembly Language inspired by the poker game Slapjack. It is a game in which player who snap the jack first wins the game (jack is the card that matches the count). displayed number match the count wins the game. The projects involves knowledge of C, Computer Organization, ARM Assembly Language, and FPGA board.

  • Designed graphic component of the game
  • Designed the functions of the game

C, ARM Assembly Language, De1-SoC Board, VGA Monitor

Joseph Yang

Slapjack, a classic poker card game, is a game that challenges player's reaction capability. Among the two players, the first person to snap the right card will win the round. However, if the player snap the wrong card, the opponent player will unconditionally win the round. Each player is given 3 hearts indicating their score status. The player who lose the round will lose 1 heart; the first player to lose all 3 hearts lose the game. The game is controlled using De1-SoC board (FPGA board), which has the following functions:

  • FPGA Keys: Dealing and snapping the card (Key 0 & 1 for Player 1; Key 2 & 3 for Player 2)

  • Each player is given 3 hearts in every game
  • Player who snap the wrong card will lose 1 heart
  • Player who snap the right card last will lose 1 heart
  • The first player to lose all 3 hearts lose the game
