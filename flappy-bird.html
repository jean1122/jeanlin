This is a project recreates the classic mobile game, Flappy Bird, using Verilog. It is a game in which the player controls a bird to fly over different obstacles. The projects involves the knowledge of Verilog, Digital System, and FPGA board.

  • Used finite state machine to design game animation
  • Designed graphic component of the game
  • Designed the functions of the game

Verilog, De1-SoC Board, VGA Monitor

Joseph Yang

The classic mobile game, Flappy Bird, is a game that player can control the bird to fly up to a certain height to avoid obstables. The game board is displayed on VGA monitor. To start, the player has to toggle the keys on FPGA board to keep the bird in certain height; otherwise, the bird will fall on its own if no action is done. In order to win the game, the player has to make the bird fly through obstacles with different height. The game is controlled using De1-SoC board (FPGA board), which has the following functions:

  • FPGA LED: Displays remaining lives of the bird
  • FPGA Keys: Controls the bird to fly up or reset the game
  • FPGA Hex Display: Displays score board
  • FPGA Switch: Controls the game's difficulty level
  • Each player is given 3 lives in each game
  • Player will lose 1 life if the bird hits any obstables, floor, or ceiling

The game consists 3 levels of difficulty: easy, medium, hard. The difficulty level depends on how fast the bird moves. To choose the level of difficulty, player can toggle switches on De1-SoC board:

  • Toggle Switch 0: Easy Level
  • Toggle Switch 0 & 1: Medium Level
  • Toggle Switch 0 & 1 & 2: Hard Level
